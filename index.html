Что такое наш Телеграм Бот?

Наш бот — это удобный помощник, который показывает расписание групп по нашему учебному направлению. Написан с использованием aiogram — современной библиотеки для разработки Telegram ботов на Python.

Он позволяет быстро получать актуальное расписание, не заходя в сложные системы, а просто написав боту в Телеграме.

Процесс создания

  1. Постановка целей — сделать удобный и быстрый доступ к расписанию через Telegram.
  2. Выбор технологий — Python + aiogram для надёжной и быстрой разработки.
  3. Разработка логики — реализация обработки команд и отображения расписания.
  4. Тестирование и исправления — чтобы бот работал корректно и без сбоев.
  5. Запуск и сопровождение — обновление расписания и добавление новых функций.

Функции бота и примеры работы

  • Показ расписания групп по направлениям.
  • Поиск расписания по дате или группе.
  • Уведомления об изменениях в расписании.
  • Интерактивное меню для удобного навигации.

Пример команды:

/schedule group_123

Бот покажет расписание для группы с ID group_123.
