Что такое наш Телеграм Бот?

Наш бот создан для студентов и преподавателей, чтобы быстро и удобно получать актуальное расписание групп по направлениям обучения прямо в Telegram.

Он написан на Python с использованием aiogram — мощной асинхронной библиотеки для создания Telegram ботов.

Почему именно наш бот?

  • Интерактивность — бот понимает разные команды и подсказывает расписание.
  • Обновление в реальном времени — расписание всегда актуально.
  • Уведомления об изменениях — не пропустите важное.
  • Простота использования — не нужно заходить в сложные сайты.

Попробуйте прямо сейчас: https://t.me/your_bot_username

Пример команды для получения расписания:

/schedule group_123

Пример Python-кода с aiogram:

from aiogram import Bot, Dispatcher, executor, types

bot = Bot(token='YOUR_TOKEN')
dp = Dispatcher(bot)

@dp.message_handler(commands=['schedule'])
async def send_schedule(message: types.Message):
    group_id = message.get_args()
    schedule = get_schedule_for_group(group_id)
    await message.reply(schedule)

if __name__ == '__main__':
    executor.start_polling(dp)
