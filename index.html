О боте

Наш Telegram бот на библиотеке aiogram показывает расписание групп по вашему направлению. Цели создания: удобный доступ к расписанию в Telegram, простота использования, актуальная информация. Бот позволяет быстро получать расписание на нужную дату и группу, используя простой интерфейс.

Процесс создания

  • Определили цели и функционал бота
  • Разработали бэкенд на aiogram для взаимодействия с Telegram API
  • Подключили базу данных с расписанием и пользователями
  • Реализовали функции запроса расписания по группе и дате
  • Создали этот сайт для удобного просмотра информации и обратной связи

Технологии

  • Python, aiogram для Telegram бота
  • JavaScript, HTML, CSS для сайта
  • Локальное хранение расписания и отзывов (mock API)
